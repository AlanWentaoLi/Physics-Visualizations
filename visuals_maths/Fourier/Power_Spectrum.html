Instructions for use

Welcome to Imperial Fourier Visualisation: Power Spectrum Part.

This is the title and name of the Fourier Series Visualisation.

You can select the function you want to visualize from here.

You can select the coefficient you want to visualize from here. It is either a_n and b_n or alpha_n and theta_n.

Here, the power spectrum of either a_n b_n or alpha_n theta_n will be displayed.

This is the Amplitude Slider that allows you to change the Amplitude.

This is the Number Slider that allows you to change the Number of Terms. When N=0, it displays a_0; when N=1, it displays a_1 and b_1

This is the Period Slider that allows you to change the Period.

Have fun!
